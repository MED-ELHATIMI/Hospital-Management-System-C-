 Hospital-Management-System-C- est une application console développée en C++ permettant de gérer les opérations essentielles d’un hôpital.
Ce projet simule un système réel de gestion hospitalière en utilisant la programmation orientée objet (POO) et la gestion de fichiers.

 . Fonctionnalités
 Gestion des patients (ajout, suppression, affichage)
 Gestion des médecins
 Gestion des rendez-vous
 Suivi des paiements
 Gestion des dossiers médicaux
 Sauvegarde des données avec fichiers
 Technologies utilisées
  
Langage : C++
Concepts : Programmation Orientée Objet (POO)
Stockage : Fichiers (File Handling)

. Structure du projet
gestion_hopital.cpp

. Objectif du projet
Ce projet a été réalisé dans le but de :
Comprendre la POO en C++
Manipuler les fichiers en C++
Simuler un système réel de gestion
